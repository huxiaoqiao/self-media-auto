# 🚀 IP 爆款制造机 (自媒体自动化工具)

> **一句话介绍：AI 帮你在后台干苦力活，你只负责最后的审美把关。**  
> 这是一套专门为自媒体人设计的自动化工具。它能帮你盯着全网的热搜内容，帮你把素材改写成有你个人风格的文章和脚本，还能自动给文章画图、发到公众号和飞书。

---

## 🛠️ 核心功能

- **🔍 找热门素材**：盯着小红书、抖音、公众号里的爆款。你可以让它搜关键词，也可以直接把文章链接或视频链接给它，它会自动抓取里面的内容。
- **🧠 自动改写内容**：
  - **视频转文字**：给它短视频链接，它能自动把里面的说话内容转成文字。
  - **深度改写**：用大模型把素材重新写一遍，去掉“AI 味”，变成像真人写的文章。
- **🎨 自动给文章配图**：
  - **自动画画**：根据文章内容，自动给文章画好封面图和插图，不用再到处找无版权图。
  - **自动排版**：画好的图会自动插进文章合适的位置，阅读体验更好。
- **📡 一键同步发布**：
  - **发到公众号**：写好的文章一键传到微信草稿箱，你在手机上点一下就能发。
  - **存到飞书**：自动化同步到飞书云文档，方便存档和团队查看。

## 🚀 快速上手

### 1. 准备环境
```bash
# 运行初始化脚本
.\setup.ps1
```

### 2. 设置你的密钥
运行下面的命令，按照提示填入你的 API Key：
```bash
python workflow_controller.py setup
```

### 3. 三步开始干活

#### 第一步：找个想写的题目
```bash
# 方法 A: 让系统帮你搜热门
python workflow_controller.py discovery --keyword "人工智能"
# 方法 B: 直接丢给它一篇文章链接
python workflow_controller.py from-article --url "文章链接"
# 方法 C: 直接丢给它一个视频链接
python workflow_controller.py from-video --url "视频链接"
```

#### 第二步：让 AI 开始改写
```bash
# 填入第一步里得到的序号，AI 就会开始改写
python workflow_controller.py repurpose --id 序号
```

#### 第三步：自动画图并发布
```bash
# 画好图并传到公众号草稿箱
python workflow_controller.py publish
```

```bash

```
