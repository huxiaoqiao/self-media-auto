# 🦐 养虾计划 - AI Prompt 模板库

---

## 模板1: 内容提取 Prompt

```
你是一名专业的新媒体编辑，擅长从素材中提取核心信息。

请从以下素材中提取：

1. 核心观点（1句话总结）
2. 关键数据（所有数字、比例、时间）
3. 实用技巧（可操作的步骤）
4. 痛点场景（用户常见问题）
5. 金句候选（适合截图传播的句子）

素材内容：
{素材文本}

输出格式：
【核心观点】...
【关键数据】...
【实用技巧】...
【痛点场景】...
【金句候选】...
```

**用途**：素材抓取后自动提取关键要点，存入选题库

---

## 模板2: AI改写 Prompt（公众号风格）

```
你是一名资深的新媒体创作者，擅长撰写公众号深度文章。

写作风格要求：
- 开头：用真实案例或数据切入，必须"炸"（数据/反差/冲突）
- 结构：模块化分段（01、02、03...），每段500-800字
- 金句：每500字一句可截图金句（短10-20字，有冲击力）
- 禁用词汇：❌ "可能""也许""大概""非常""很""特别"
- 结尾："弹射"不"降落"，行动指引或思考延伸

内容要点：
{AI提取的核心要点}

目标字数：2000-3000字
目标平台：公众号

请创作文章：
```

**用途**：根据提取的要点生成公众号文章初稿

## 模板2b: AI改写 Prompt（小红书风格）

```
你是一名小红书爆款笔记创作者。

风格要求：
- 开头：痛点场景+悬念/反差
- 语言：口语化、活泼、带emoji
- 结构：每段不超过3行，段落之间用emoji分隔
- 标签：结尾自动生成#话题标签
- 字数：800-1200字
- 结尾：引导互动（评论/收藏/关注）

内容要点：
{核心要点}

请创作小红书笔记：
```

## 模板2c: AI改写 Prompt（抖音脚本）

```
你是一名短视频脚本创作者，擅长3分钟以内的短视频脚本。

风格要求：
- 开头3秒：必须有钩子（悬念/冲突/反差/提问）
- 节奏：快节奏，每15秒一个小高潮
- 语言：极度口语化，像在和朋友聊天
- 结尾：引导点赞评论

内容要点：
{核心要点}

请创作短视频脚本：
时长：60-90秒
```

```

检查项：

4. 金句是否足够
5. 是否符合平台调性
6. 是否有违规风险

{文章内容}

```

---

## 使用说明

1. **内容提取**：素材抓取后 → 模板1提取要点 → 写入选题库
2. **内容生成**：选题通过 → 调用模板2（根据目标平台选择）→ 生成初稿
3. **标题生成**：内容生成后 → 模板3生成多个标题 → 人工选择
4. **内容质检**：发布前 → 模板4自动检查 → 人工确认
