# OpenClaw+微信保姆级部署教程，养小龙虾的最佳方式！

**作者**: 公子龙
**发布时间**: 2026年3月24日 09:57

---

养龙虾浪潮最近是火的不行，全网都在装 龙虾（OpenClaw），但是对于普通用户来说，怎么安装龙虾openclaw,以及如何使用openclaw，都还是有一定的门槛的，并且还需要自己在电脑上进行操作，无法做到像日常对话一样丝滑地指挥龙虾工作。

但是随着最近腾讯的一次大动作，现在通过微信指挥龙虾干活不再是想象了，已经成为现实！

3月22日腾讯正式推出 微信「ClawBot」插件，支持接入 OpenClaw。相当于把「专属私人AI助理」直接装进微信里。很多人还不知道怎么装、在哪装、能不能成功，今天我把完整流程、三种安装方案、微信插件绑定一次性讲清楚，照着做，你也能拥有自己的私人AI。

首先我们得搞懂：龙虾是什么？简单说：龙虾 = OpenClaw = 你的私人AI服务端，你可以把它部署在本地、云服务器，再通过微信插件连接，实现微信内直接调用AI。 

一句话总结：先装龙虾 → 再装微信插件 → 绑定 → 成功拥有AI助理。

龙虾（OpenClaw）3种安装方案，任选一种，按需选择即可。

① 本地电脑安装（零成本、新手首选，超详细版本）

适合只想体验、不想买服务器的人。这种方式最灵活，最方便，在这里我将从0开始介绍如何安装openclaw,并且对接微信通道。

openclaw官网是这个：


https://docs.openclaw.ai/zh-CN

windows怎么安装openclaw呢？先执行这个命令：

iwr -useb https://openclaw.ai/install.ps1 | iex

然后启动openclaw服务：


openclaw onboard --install-daemon

紧接着就是不断按回车下一步，配置对应的大模型API等，这里不再过多说明了。安装成功后检查：

openclaw status

需要特别注意的是：直接在Powershell安装微信的openclaw插件会出现这个问题：找不到安装的openclaw,原因是环境变量为空!

那么这个大BUG怎么解决呢？？？这可有点难搞，突然灵机一动，我想了一个好办法，利用git的Bash环境来解决这个问题，下载Git工具。然后提前给系统变量添加npm的安装路径，保存全局应用。
地址：

https://git-scm.com/install/
这个时候再打开Git bash窗口，在这个窗口下执行：
npx -y @tencent-weixin/openclaw-weixin-cli@latest install


惊喜来了，安装成功了！然后就会提醒你用「微信扫描绑定」。




紧接着就手机微信会弹出「当前已经连接一个openclaw」,因为我之前连了腾讯云的openclaw,断开当前连接建立新的连接。


然后，重新启动：

openclaw gateway --port 18789
成功绑定微信之后，我们可以通过openclaw的web控制台进行查看，已经配置的通道。通过以下命令生成带令牌的URL路径：
$ openclaw dashboard --no-open




OK,终于大功告成了，我们来看看效果：
让他介绍一下自己，它的名称在安装的时候可以设置，你也可以让openclaw帮你修改。
然后我们试着，让他在桌面创建一个工作计划的文件，非常棒,可以看到成功创建了工作计划.md文件，你知道这意味着什么吗？以后你也可以通过微信，直接在手机指挥你的电脑干活了。


现在我们再看第二种云服务器部署的方案：

② 腾讯云 Lighthouse部署（云端虾🦞 24小时在线）

这种方式将openclaw部署在云端，基于腾讯云的生态和微信无缝衔接，非常适合想要快速养龙虾的朋友，简单方便，容易操作。就重点说说这种方式。

地址：

https://cloud.tencent.com/act/pro/openclaw

首先登陆腾讯云openclaw的快速通道，购买对应的openclaw轻量级服务器，通过这种方法购买的云服务器会直接一键部署了openclaw的镜像，大大降低了普通人操作的门槛。

好了，现在我们安装好了腾讯云服务器的openclaw,现在需要进行配置微信授权

授权成功之后，还要对模型进行配置，才能够使用：

强烈建议看看这个教程，轻量云OpenClaw接入微信指南：
https://cloud.tencent.com/developer/article/2643772
