📝 实际应用场景
🌐 场景一：个人博客域名

用 Hexo 或 Hugo 搭建的博客，部署在某台服务器上。之前可能用 IP 访问，现在可以注册一个短域名，添加 A 记录指向服务器 IP，配合 Nginx 反向代理实现自定义域名访问。

🌐 场景二：开源项目演示

开源项目需要给用户一个演示地址时，注册一个短域名，CNAME 到 GitHub Pages 或 Vercel 地址，好记又专业。

🌐 场景三：API 调试

调试第三方 API 需要回调地址时，注册一个域名，添加 A 记录指向服务器，配合 ngrok 或内网穿透工具，实现 HTTPS 回调。
